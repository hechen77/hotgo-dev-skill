## 6.1 环境与项目创建

### 环境要求

| 工具 | 版本 | 验证命令 |
|------|------|---------|
| Flutter SDK | >= 3.11.5（含 Desktop 支持） | `flutter config --enable-windows-desktop` |
| Visual Studio 2022 | 含"使用 C++ 的桌面开发"工作负载 | — |
| Windows SDK | >= 10.0.19041 | — |
| macOS（如要构建 mac 端） | Xcode 15+ | — |
| Linux（如要构建 linux 端） | GTK 3.0+, CMake | — |

### 创建项目

```bash
# 1. 确认桌面端已启用
flutter config --enable-windows-desktop
flutter config --enable-macos-desktop
flutter config --enable-linux-desktop

# 2. 验证支持
flutter doctor

# 3. 创建项目
flutter create --org com.yourcompany --project-name your_desktop_app .

# 4. 安装依赖
flutter pub get

# 5. 运行到 Windows（默认桌面设备）
flutter run -d windows

# 6. 运行到 macOS
flutter run -d macos

# 7. 运行到 Linux
flutter run -d linux

# 8. 构建 Windows exe
flutter build windows

# 9. 构建 macOS app
flutter build macos

# 10. 构建 Linux
flutter build linux
```

### 桌面端独有依赖详解

| 依赖 | 为什么桌面端需要 | 移动端没有的原因 |
|------|----------------|---------------|
| `window_manager` | 控制窗口大小、位置、最大最小化、`preventClose`（关闭→隐藏托盘） | 移动端全屏运行，无窗口概念 |
| `tray_manager` | 系统托盘图标 + 菜单 + 点击交互 | 移动端无托盘 |
| `forui` | 桌面风格 UI 组件（侧边栏、表格、对话框、工具栏），更贴近桌面交互习惯 | 移动端用 TDesign Flutter |
| `syncfusion_flutter_datagrid` | 类 Excel 大数据量表格（冻结列、排序、筛选） | 移动端屏幕小，不需要复杂表格 |
| `restart_app` | 登出后重启应用回到登录页 | 移动端用路由跳转即可 |

```dart
import 'dart:io';
import 'package:flutter/material.dart';
import 'package:forui/forui.dart';
import 'package:get/get.dart';
import 'package:window_manager/window_manager.dart';

import 'package:your_app/routers/router.dart';
import 'package:your_app/service/auth.dart';
import 'package:your_app/service/user.dart';
import 'package:your_app/service/windows_manager.dart';

import 'package:your_app/service/ui_control_service.dart';
import 'package:your_app/components/custom_title_bar/custom_appbar.dart';
import 'package:your_app/components/layout/sidebar/custom_sidebar.dart';
import 'package:your_app/components/loading_controller/loading_controller.dart';
import 'package:your_app/utils/notification/location_notifications.dart';
import 'package:your_app/utils/ws/webstocket_manage.dart';

// 开发环境：允许自签名证书
class HttpOverridesForDev extends HttpOverrides {
  @override
  HttpClient createHttpClient(SecurityContext? context) {
    return super.createHttpClient(context)
      ..badCertificateCallback = (X509Certificate cert, String host, int port) => true;
  }
}

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  HttpOverrides.global = HttpOverridesForDev();

  // 1. 初始化通知
  await NotificationHelper().initialize();

  // 2. ⭐ 初始化 Service（顺序不能乱）
  await Get.putAsync(() => AuthService().init());
  await Get.putAsync(() => UserService().init());
  await Get.putAsync(() => WebSocketManager().init());
  await Get.putAsync(() => WindowsManagerService().init());
  await Get.putAsync(() => TrayService().init());
  await Get.putAsync(() => UiControlService().init());

  // 3. ⭐ 初始化 window_manager
  await WindowManager.instance.ensureInitialized();
  const windowOptions = WindowOptions(
    center: true,
    titleBarStyle: TitleBarStyle.hidden,     // ⭐ 隐藏原生标题栏，显示自定义标题栏
  );
  windowManager.waitUntilReadyToShow(windowOptions, () async {
    await windowManager.show();
    await windowManager.focus();
    await windowManager.setIcon("assets/app_icon.ico");
    await windowManager.setHasShadow(true);
  });

  runApp(const MyApp());
}

class MyApp extends StatefulWidget {
  const MyApp({super.key});
  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> with WindowListener {

  void initState() {
    super.initState();
    windowManager.addListener(this);
  }

  // ⭐ 关闭窗口 → 隐藏到托盘（不退出）

    await windowManager.hide();

  @override
  void onWindowMaximize() =>
      Get.find<WindowsManagerService>().isMaximized.value = true;

  @override
  void onWindowUnmaximize() =>
      Get.find<WindowsManagerService>().isMaximized.value = false;

  @override
  void dispose() {
    windowManager.removeListener(this);
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return GetMaterialApp(
      navigatorKey: Get.key,
      navigatorObservers: [RouteObserverService()],
      debugShowCheckedModeBanner: false,
      title: "你的桌面应用",

      getPages: AppPages.routes,

          backgroundColor: Colors.white,
          surfaceTintColor: Colors.white,

      builder: (context, child) {
        return FBasicTheme(
          data: FThemeData(
```

## 6.7 窗口管理 Service

`lib/service/windows_manager.dart`：

```dart
import 'dart:ui';
import 'package:flutter/cupertino.dart';
import 'package:get/get.dart';
import 'package:your_app/service/ui_control_service.dart';
import 'package:window_manager/window_manager.dart';

class WindowsManagerService extends GetxService {
  bool isLock = false;

  Future<WindowsManagerService> init() async {
    await windowManager.setMinimumSize(const Size(320, 480));
    return this;
  }

  /// ⭐ 锁定登录窗口（小窗口，不可调整）
  Future<void> lockLoginSize() async {
    isLock = true;
    await windowManager.setSize(const Size(320, 480));
    await windowManager.setMinimizable(false);
    await windowManager.setMaximizable(false);
    await windowManager.setResizable(false);
    await windowManager.setTitleBarStyle(TitleBarStyle.hidden);
    await windowManager.setPreventClose(false);  // 登录页可以直接关闭
    await windowManager.center();
    Get.find<UiControlService>().hideSidebar();
  }

  /// ⭐ 解锁为主窗口（大窗口，可自由调整）
  Future<void> unlockMainSize() async {
    isLock = false;
    await windowManager.setMinimizable(true);
    await windowManager.setMaximizable(true);
    await windowManager.setResizable(true);
    await windowManager.setSize(const Size(1200, 800));
    await windowManager.setMinimumSize(const Size(1000, 600));
    await windowManager.setPreventClose(true);   // ⭐ 关闭 = 隐藏托盘
    await windowManager.center();
    Get.find<UiControlService>().showSidebar();
  }

  var isMaximized = false.obs;

  Future<void> toggleMax() async {
    if (await windowManager.isMaximized()) {
      await windowManager.unmaximize();
      isMaximized.value = false;
    } else {
      await windowManager.maximize();
      isMaximized.value = true;
    }
  }

  Future<void> minimize() async => await windowManager.minimize();
  Future<void> close() async => await windowManager.close();

  var isCloseHover = false.obs;
  void setCloseHover(bool v) => isCloseHover.value = v;

  @override
  void onClose() { super.onClose(); }
}
```

### 关键设计：登录/主窗口双尺寸

```
未登录 → lockLoginSize()
    ├── 320×480 固定大小
    ├── 不可最大/最小化
    ├── 不可拖拽调整
    ├── 关闭按钮 = 真正退出
    └── 隐藏侧边栏

登录成功 → unlockMainSize()
    ├── 1200×800（可拖拽调整）
    ├── 最小 1000×600
    ├── 关闭按钮 = 隐藏到托盘
    ├── 显示侧边栏
    └── 显示自定义标题栏
```

## 6.8 系统托盘 Service

`lib/service/tray.dart`：

```dart
import 'package:flutter/cupertino.dart';
import 'package:get/get.dart';
import 'package:tray_manager/tray_manager.dart';
import 'package:window_manager/window_manager.dart';

class TrayService extends GetxService with TrayListener {
  Future<TrayService> init() async {
    await trayManager.setIcon('assets/app_icon.ico');
    await trayManager.setToolTip("你的桌面应用");
    await trayManager.setContextMenu(Menu(items: [
      MenuItem(key: 'show', label: '显示'),
      MenuItem(key: 'exit', label: '退出'),
    ]));
    trayManager.addListener(this);
    return this;
  }

  /// 点击托盘图标 → 切换显示/隐藏
  @override
  void onTrayIconMouseDown() async {
    if (!await windowManager.isVisible()) {
      await windowManager.show();
      await windowManager.focus();
    } else {
      await windowManager.hide();
    }
  }

  /// 右键托盘 → 弹出菜单
  @override
  void onTrayIconRightMouseDown() async {
    trayManager.popUpContextMenu();
  }

  /// 菜单项点击
  @override
  void onTrayMenuItemClick(MenuItem item) async {
    switch (item.key) {
      case 'show':
        await windowManager.show();
        await windowManager.focus();
        break;
      case 'exit':
        await windowManager.setPreventClose(false); // 允许关闭
        await windowManager.close();                 // 真正退出
        break;
    }
  }

  @override
  void onClose() { super.onClose(); }
}
```

### 托盘工作流

```
用户点击关闭按钮 → onWindowClose() → windowManager.hide()
应用仍在后台运行，WS 连接保持

用户点击托盘图标 → onTrayIconMouseDown() → toggle 显示/隐藏
用户右键托盘 → onTrayIconRightMouseDown() → 弹出菜单
点击"退出" → onTrayMenuItemClick('exit') → 真正关闭应用
```

```dart
import 'package:get/get.dart';

class UiControlService extends GetxService {
  Future<UiControlService> init() async => this;

  // ⭐ 侧边栏状态
  var sidebarVisible = true.obs;
  void showSidebar() => sidebarVisible.value = true;
  void hideSidebar() => sidebarVisible.value = false;

  // ⭐ 侧边栏宽度（可折叠/展开）
  double defaultSidebarWidth = 60.0;
  var sidebarWidth = 60.0.obs;
  void expandSidebar() => sidebarWidth.value = 150.0;
  void collapseSidebar() => sidebarWidth.value = defaultSidebarWidth;
  void setSidebarWidthToZero() => sidebarWidth.value = 0;
  void toggleSidebarWidth() {
    sidebarWidth.value == defaultSidebarWidth
        ? expandSidebar() : collapseSidebar();
  }

  // ⭐ 标题栏状态
  var titleBarVisible = true.obs;
  void showTitleBar() => titleBarVisible.value = true;
  void hideTitleBar() => titleBarVisible.value = false;

  // ⭐ 标题栏文字
  var titleBarText = "你的桌面应用".obs;
  void setTitleBarText(String text) => titleBarText.value = text;

  @override
  void onClose() { super.onClose(); }
}
```

```dart
List<Widget> customRightButtons({

            child: Icon(
              window.isMaximized.value ? FIcons.minimize : FIcons.maximize,

            ),
          ),
        ),
      )),
    if (close)
      Obx(() => MouseRegion(
        onEnter: (_) => window.setCloseHover(true),
        onExit: (_) => window.setCloseHover(false),
        child: Material(
          color: Colors.transparent,

            mouseCursor: SystemMouseCursors.click,

  ];
```

## 6.12 路由系统

与移动端基本一致，桌面端额外增加了侧边栏和标题栏的联动：

```dart
class RoutePaths {
  static const String home = '/';
  static const String login = '/login';
  static const String dashboard = '/dashboard';
  static const String orderList = '/customer/list';
  // ...更多路由
}

// ⭐ 哪些页面显示侧边栏
const showSidebarPage = [
  RoutePaths.home,
  RoutePaths.dashboard,
  RoutePaths.orderList,
  RoutePaths.settings,
];

// ⭐ 页面标题映射
const pageTitles = {
  RoutePaths.home: "首页",
  RoutePaths.dashboard: "仪表盘",
  RoutePaths.orderList: "订单列表",
};

// ⭐ 路由观察者自动更新标题栏
class RouteObserverService extends GetObserver {
  @override
  void didPush(Route route, Route? previousRoute) {
    _sync(route);
  }

  void _sync(Route? route) {
    WidgetsBinding.instance.addPostFrameCallback((_) {
      final name = route?.settings.name;
      if (name == null || name == RoutePaths.login) return;
      final ui = Get.find<UiControlService>();
      if (showSidebarPage.contains(name)) {
        ui.collapseSidebar();              // 主页面显示侧边栏
      } else {
        ui.setSidebarWidthToZero();        // 详情页隐藏侧边栏
      }
      ui.setTitleBarText(pageTitles[name] ?? "你的桌面应用");
    });
  }
}
```

## 6.14 桌面端特有模式

### 6.14.1 登录 → 主页切换流程

```
App 启动 → 检查登录状态
  ├── 未登录 → lockLoginSize() → LoginPage（320×480 小窗口）
  │     └── 登录成功 → unlockMainSize() → HomePage（1200×800 主窗口）
  └── 已登录 → unlockMainSize() → HomePage

用户登出 → 重启应用（restart_app）
```

### 6.14.2 窗口控制

| 操作 | 行为 | API |
|------|------|-----|
| 点关闭按钮 | 隐藏到托盘 | `onWindowClose() → windowManager.hide()` |
| 点托盘图标 | 切换显示/隐藏 | `onTrayIconMouseDown()` |
| 双击标题栏 | 最大化/还原 | `onDoubleTap → toggleMax()` |
| 拖拽标题栏 | 移动窗口 | `onPanStart → windowManager.startDragging()` |
| 托盘退出 | 真正关闭应用 | `setPreventClose(false) → close()` |

### 6.14.3 鼠标交互

桌面端需要显式设置鼠标样式：

```dart
MouseRegion(
  cursor: SystemMouseCursors.click,
  child: /* 可点击的元素 */,

// Forui 按钮自带 click 样式
FButton(onPress: () {}, child: Text("按钮"))
```

## 6.17 与移动端的核心差异

| 特性 | 移动端 | 桌面端 |
|------|--------|--------|
| **UI 框架** | TDesign Flutter 为主 | Forui 为主 + TDesign 辅助 |
| **窗口管理** | 无（全屏） | `window_manager`：最大化/最小化/锁大小/隐藏托盘 |
| **系统托盘** | 无 | `tray_manager`：图标 + 菜单 + 点击交互 |
| **标题栏** | 系统默认 AppBar | 自定义 `CustomTitleBar`（拖拽 + 窗口按钮） |
| **导航** | BottomNavigationBar | 左侧侧边栏（可折叠） |
| **关闭行为** | 直接退出 | 隐藏到托盘 → 真正退出通过托盘菜单 |
| **登录页** | 全屏 | 320×480 锁定小窗口 |
| **鼠标交互** | 无（触摸） | 必须设置 `MouseRegion` / `SystemMouseCursors.click` |
| **进程守护** | `flutter_foreground_task` | 不需要（桌面应用不会被系统杀掉） |
| **后台重连** | ForegroundTask `onRepeatEvent` | 不需要（WS 连接不会被切断） |
| **API 前缀** | `/home` | `/api` |
| **构建产物** | APK / IPA | .exe / .app / Linux 二进制 |
| **图标格式** | PNG (adaptive icon) | .ico (Windows) / .icns (macOS) |
